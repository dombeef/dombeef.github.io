GlowBoard

Project Description

Created to use cheap usb battery packs as hotswappable batteries, this project was started in mind to make a modular light setup for any length longboard or skateboard.

It uses a cheap programmable LED strip based on the WS2812B LEDS and an ATTINY85 as the processor for handling controlling the LEDS among keeping track of the speed of the board from a hall effect sensor.

The project is currently on hold until I can figure out a safe way to waterproof the whole system and the battery packs as well as replacing the WS2812B LEDs with something more reliable, since if a single LED breaks, it causes the rest of the LEDs in the set to fail to function.
