Help

To use this tool:

  • You need a Hanshow ESL or OEPL compatible E Ink tag with custom firmware flashed to it. That can be done by opening the device, soldering a USB serial device to certain test pads, and then flashing a custom BIN firmware via the original website here: https://atc1441.github.io/ATC_BLE_OEPL_Image_Upload.html
  • Connect to the device over BLE.
  • Select an image or choose the default.
  • Make sure to choose the correct output resolution based on your ESL tag.
  • Confirm that your ESL tag works in 3-color mode, otherwise switch the color mode to black/white.
  • Click "Upload Compressed" to start the upload to the ESL!

About

This webpage was created by dombeef, based on and referencing the original image conversion / BLE transmission code by atc1441 on Github.

This page was originally designed to work with Pricer ESL tags and communicate to them with custom hardware. I stripped that transmission code and adapted it to work over Bluetooth with Hanshow ESL tags (potentially others with the same OpenEpaperLink firmware flashed to them!).

Special thanks to atc1441 for their excellent research, documentation, and open-source tools to make this possible. Link to their webpage: https://atc1441.github.io/ATC_BLE_OEPL_Image_Upload.html
